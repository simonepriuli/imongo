# IMongo

IMongo is a utility package that simplifies the always frustrating process of connecting to MongoDB in TypeScript.

## Authors

- [@simonepriuli](https://www.github.com/simonepriuli)

## Installation

Install IMongo with npm

```bash
  npm i imongo@latest
```
